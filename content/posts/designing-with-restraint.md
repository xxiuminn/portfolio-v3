Minimal layouts only work when the spacing, typography, and rhythm carry real intention.

In the reference that inspired this project, the screen is quiet because very little competes for attention. That quietness is useful, but it becomes fragile when you move from a single mobile layout to a full responsive site.

## What changed here

Instead of stretching the mobile design, the desktop version introduces:

- wider content framing
- multi-column project and journal grids
- more deliberate contrast between the hero and supporting sections

## Why this matters

A portfolio should feel calm, but it should still use the available screen. Empty space is valuable only when it improves reading, hierarchy, or mood.
